#### Step 4: Add an Image to README

Let's make it visual! First, we need an image to add.

👉 **Account to use**: `Tony405-spec`

**Option A: Use an existing image from your computer**
1. Find any image on your computer (screenshot, logo, etc.)
2. Rename it to `github-workflow.png` (or similar)

**Option B: Create a simple diagram**
1. Go to https://app.diagrams.net/
2. Create a simple flowchart showing: `Code → Push → PR → Review → Merge`
3. Export as PNG, save as `github-workflow.png`

**Now add it to GitHub:**

1. From your repository main page, click **"Add file"** dropdown → **"Upload files"**
2. Drag and drop your image file
3. In the commit message: `Add workflow diagram image`
4. Click **"Commit changes"**
